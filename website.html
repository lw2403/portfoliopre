USABILITY ISSUES

Analyze the current usability issues

I identfiied the current website's problems through evaluating its usability, memoriability, and learnability issues. Below are the breakdown analysis with images.

Usability

Problem Summary: First, the text and pictures are unaligned making it hard to read. Second, the picture loop used to show recommended food is distracting and inefficient because users cannot compare top selling items side by side. Additionally, the button states are unclear and confusing. Users will think they have activated a button when they haven‚Äôt.

Solution:I got rid of the picture loop, created a style sheet with different button states, and alligned all pictures and texts to improve readability

ACCESSIBILITY ISSUES

Identify current accessibility issues
Wave Report
Analysis

The website‚Äôs form control misses text label, so the function or purpose of that form control will not be presented to screen reader users. Navigating to a button that misses descriptive text will cause screen reader users to be confused by the function of the button. If a link contains no text, the function of the link will confusing to keyboard and screen reader users. The target for the ARIA reference does not exist. ARIA labels and descriptions will not be presented if the element referenced does not exist in the page.Very low colour contrast between background & text.The contrast of text is especially important for users with low vision.

üëâüëâüëâSOLUTION

I added form labels, button texts, colour contrasts, and fixed ARIA reference.
